# Ryzen Hackintosh Patcher

This script will patch your fresh 10.12 install to run on AMD systems.

### Using the Patcher:

```sh
$ git clone https://github.com/jaytarang92/ryzenPatcher.git
$ chmod +x ryzenPatcher.sh
$ sudo ./ryzenPatcher.sh
```
